Welcome to my guitar Auto Tabber!

Enter the notes below, in GUITAR FORMAT (see example input below).
Please note that the only reason why guitar format is required is because it is the most confortable format to test the app; the app does not 'cheat'.
Once you input the notes, click the "Submit" button to run the Auto Tabber.
The Auto Tabber will try to find a decent arrangement for the given score on guitar. Furthermore, it will try to split the notes into phrases for clarity.
The Auto Tabber uses a Probabilistic Graphical Model (more precisely, a Hidden Markov Model) to find the best arragement it can.
Please note, I only spent two days working on this project, it doesn't have much testing. It might do weird things!
Check the code at: https://github.com/sprillo/web-app/blob/master/autoTabber.py
NOTE: When several notes are played at the same time, input them in order from lowest to highest string.
